# 🧠 LLM Reasoning Leaderboard

A project exploring how different Large Language Models (LLMs) perform on reasoning tasks using real-world data and evaluation techniques.

## 📌 Overview

This project evaluates the **reasoning capabilities of LLMs** by:

- Fetching and processing real-world data (e.g., news)
- Running structured prompts on LLMs
- Comparing outputs across models
- Building a simple leaderboard-style evaluation

The goal is to understand how well LLMs handle:
- Logical reasoning
- Context understanding
- Response consistency
